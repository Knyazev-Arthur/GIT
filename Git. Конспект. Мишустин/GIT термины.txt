Git - программа, следящая за изменениями всех файлов на ПК
GitHub - веб-приложение, позволяющее загрузить в облако все изменения моих файлов.

Репозиторий - папка на ПК, изменения в которой отслеживает Git.
Локальный репозиторий - папка на ПК.
Удаленный репозиторий - папка в облаке.

Ветка Git - последовательность коммитов, имеющая опеределнная имя
